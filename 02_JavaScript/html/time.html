시간 지연 함수

모두 비동기로 동작

setTimeout

지연 시간이 지난 후 1회만 수행

window. 는 생략 가능

window.setTimeout(함수, 지연시간(ms));

setTimeout(() => { 여기에 내용 }, 지연시간(ms));

setInterval set

지연 시간이 지날 때 마다 계속 수행

window. 는 생략 가능

window.setInterval(함수, 지연시간(ms));

const 변수명 = setInterval(() => { 여기에 내용 }, 지연시간(ms));

setInterval clear

setInterval()로 인해 생성된 인터벌을 삭제하는 함수

window. 는 생략 가능

window.clearInterval(생성된 인터벌);

setInterval(변수명);

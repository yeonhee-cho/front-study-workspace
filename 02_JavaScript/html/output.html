alert

별도의 대화 상자

확인 버튼만 존재

window. 는 생략 가능

window.alert(내용(문자열/변수명/숫자형 등));

alert("이것은 alert 1번 내용입니다.");

alert(`${변수명} 입니다.`);

prompt

별도의 대화 상자

alert창에서 텍스트를 입력할 수 있는 인풋 존재

window. 는 생략 가능

window.prompt(내용(문자열/변수명/숫자형 등));

prompt("이것은 prompt 1번 내용입니다.");

prompt(`${변수명} 입니다.`);

별도의 대화 상자

log

F12 (개발자 도구) 콘솔을 통해 데이터를 출력
